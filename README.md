# Blockchain-Based-Secure-Key-Exchange-Framework-Using-the-Diffie-Hellman-Protocol
Final year project proposes a secure real-time key exchange system using Diffie–Hellman and blockchain technology. Public keys are verified through blockchain to prevent tampering and man-in-the-middle attacks, while the shared secret key enables secure encryption. Built with Python and Flask, the system supports secure messaging and file sharing.
Abstract:The rapid growth of digital communication, cloud computing, Internet of Things (IoT), healthcare systems, and enterprise networks has significantly increased the need for secure real-time data transmission. Traditional communication systems often face serious security threats such as eavesdropping, key interception, unauthorized access, and man-in-the-middle (MITM) attacks due to dependence on centralized authentication mechanisms and insecure key exchange methods. To address these challenges, this project proposes a Blockchain-Based Secure Key Exchange Framework Using the Diffie–Hellman Protocol for Real-Time Data Transmission. The proposed framework integrates the Diffie–Hellman key exchange algorithm with blockchain technology to create a decentralized, tamper-proof, and secure communication environment. Diffie–Hellman enables two communicating users to generate a common shared secret key without directly transmitting the key over the network, thereby reducing interception risks. ensuring authenticity and preventing unauthorized modification. The system is implemented using Python, Flask, lightweight blockchain architecture, and cryptographic libraries such as PyCryptodome. Flask-based RESTful APIs facilitate secure communication between users, while symmetric encryption is used to protect transmitted messages and files after successful key generation. The blockchain module maintains transaction integrity through cryptographic hashing and decentralized verification. Experimental results demonstrate that the proposed framework successfully prevents public key tampering, mitigates MITM attacks, ensures confidentiality and integrity, and provides secure encrypted communication with minimal latency. The lightweight design makes the framework suitable for IoT devices, cloud platforms, healthcare applications, enterprise communication, and secure messaging systems. The proposed solution offers a scalable, decentralized, and practical approach for modern cybersecurity challenges in real-time communication.
